## FirePress Casper

FirePress Casper is fork of the Ghost Theme 'Casper'. We decided to make it available to everyone and open sourced here: https://github.com/firepress-org/FirePress_Casper

## Features: 

- Table (grid)
- Icons (font-awesome)
- Buttons
- Code Syntax (highlight) 
- and more to come
